字符串相乘

Leetcode.python字符串相乘

题目

给定两个以字符串形式表示的非负整数 num1 和 num2，返回 num1 和 num2 的乘积，它们的乘积也表示为字符串形式。

示例 1:

输入: num1 = “2”, num2 = “3”
输出: “6”
示例 2:

输入: num1 = “123”, num2 = “456”
输出: “56088”
说明：

num1 和 num2 的长度小于110。
num1 和 num2 只包含数字 0-9。
num1 和 num2 均不以零开头，除非是数字 0 本身。
不能使用任何标准库的大数类型（比如 BigInteger）或直接将输入转换为整数来处理。

代码

class Solution:
def multiply(self, str1: str, str2: str) -> str:
sum1=0
sum2=0
if len(str1)<=110 or len(str2)<=110:
if int(str1[0])==0 or int(str2[0])==0:
return “0”
else:
for i in range(len(str1)):
sum1=int(str1[i])pow(10,len(str1)-i-1)+sum1
for i in range(len(str2)):
sum2=int(str2[i])pow(10,len(str2)-i-1)+sum2
return str(sum1*sum2)
else:
return “”
